# C(AI)DENCE Documentation

C(AI)DENCE is a comprehensive AI-powered marketing intelligence suite that combines autonomous campaign management, content generation, influencer discovery, and analytics in a single unified platform. It is built on a FastAPI backend with a Next.js 14 frontend and supports multi-tenancy across a four-tier hierarchy: Platform → Agency → Brand → Creator.
